# PromptForge Dev

Bản extension được chỉnh lại theo UI developer prompt studio:
- Dark, technical, focused
- 4 tab: Write, Templates, Library, Refine
- Prompt health scoring
- Quick fields và smart refine actions
- Saved prompt library bằng chrome.storage
- Options page để chỉnh target model, stack, tone và review rules

## Cài đặt
1. Mở `chrome://extensions`
2. Bật **Developer mode**
3. Chọn **Load unpacked**
4. Trỏ tới thư mục này
5. Bấm icon extension để mở side panel

## Chức năng chính
- **Generate**: build prompt từ Role / Task / Stack / Context / Constraints
- **Refine**: chuyển prompt hiện tại sang tab refine và tạo prompt tinh chỉnh
- **Templates**: nạp prompt mẫu cho code review, bug investigation, refactor, API design
- **Library**: lưu prompt vào local storage và nạp lại nhanh
- **Settings**: tùy chỉnh DNA dev prompt của bạn
